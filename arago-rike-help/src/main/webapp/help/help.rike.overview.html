Rike Overview

Purpose

List, create and filter tasks, milestones and artifacts, choose tasks for manipulation.

Understanding the Overview Portlet

The first column is the task id, it's main identifier. The second column is the task status, whereas gray is 'not evaluated', red is 'not started', orange is 'in progress' and green is 'done'. The third column is the priority followed by the task title.

Tasks can be filtered by various aspects.

Creating an Task

Pushing the "plus"-Button you can create a new task.

Creating a Milestone

Pushing the "plus"-Button you can create a new milestone.

Creating an Artifact

Pushing the "plus"-Button you can create a new artifact.
