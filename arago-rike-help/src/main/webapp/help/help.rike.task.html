Task

Purpose

Manage tasks, milestones and artifacts.

Understanding the Task Portlet

With no task selected, the portlet is empty. New parts can be created by clicking 'new'. A task can be selected by clicking on a task in other portlets, like Overview or My Tasks.

Creating an Artifact

The artifact must have a name, shortname and should have a URL pointing to further information about the artifact.

Creating a Milestone

The milestone must a name and should have a URL pointing to further information about the milestone. Furthermore an end-date can be assinged and a release may be selected.

Working with Tasks

Creating a Task

The task must have name and must have a URL pointing to further information about the task. An artifact must be selected. The task will be assigned to the artifact. The size is the estimation of the LOC (Lines of Code) that need to be done. After creating a new task, the task must be evaluated.

Evaluating a Task

While evaluating you can choose the milestone of the task. Furthermore a priority and a complexity must be set. The size of the task may also be adjusted. After evaluating a task, it can either be started by someone else or yourself after 24 hours. A task may be evaluated again if it is still open.

Starting a Task

Start a task by selecting an open one, and click start.

Finishing a Task

By selecting a task, you are currently working on, you may close the task. You must enter the time (h) you put into the task and the LOC you produced while working on it.
